Информация о текстурах:

Для улучшения дизайна сайта, рекомендуется добавить следующие текстуры:

1. hero-bg-overlay.png - полупрозрачная текстура для параллакс-эффекта в героической секции
2. pattern-dots.png - текстура с точечным узором для фоновых элементов
3. pattern-grid.png - текстура с сеткой для декоративных элементов
4. wave-pattern.svg - волнистый узор для разделителей секций

Текстуры должны быть в высоком разрешении (минимум 1920x1080px) и оптимизированы для веба.
Для наилучшего эффекта используйте полупрозрачные текстуры с альфа-каналом в формате PNG.

Если у вас нет готовых текстур, вы можете скачать бесплатные наборы с сайтов:
- https://www.transparenttextures.com/
- https://www.toptal.com/designers/subtlepatterns/
- https://www.heropatterns.com/
